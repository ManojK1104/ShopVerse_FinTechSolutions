# **FinTech Analysis Project - By ShopVerse FinTech Solutions Pvt Ltd**

---

## Dataset Overview

This dataset represents a large-scale simulation of financial transactions and business operations within the Indian FinTech ecosystem.

```
Total Records: 598,789
Total Columns: 55+
Time Period: 2020 – 2025
Region: India

Note: The dataset captures multi-dimensional data including transactions, customers, lending, fraud, funding, engagement, and operational performance.

Time Intelligence

transaction_id: Unique transaction identifier
date: Transaction date
year / quarter / month / week / day: Time breakdown
day_of_week: Name of the day
is_weekend: Weekend indicator
financial_year: Financial year format

Company Intelligence

company_name: FinTech company name
company_type: Business type (NBFC, Lending, etc.)
hq_city / country: Location details
company_founded_year: Year established
total_employees: Workforce size
market_share_percent: Market share
active_users: Total active users

Transaction Details

transaction_amount: Transaction value
transaction_type: Payment mode
transaction_status: Success/Failure/Pending
payment_channel: App/Web/POS
device_type: Mobile/Desktop
merchant_category: Business category
transaction_currency: Currency (INR)
cross_border_flag: International transaction indicator

Financial Metrics

revenue / profit: Earnings
processing_fee / commission_fee: Charges applied
gst_amount: Tax component
net_revenue: Final revenue
cost_of_acquisition: Customer acquisition cost

Customer Intelligence

customer_id: Unique customer ID
customer_age / age_group / gender: Demographics
customer_segment: Retail/SME/Corporate
customer_region / city: Location
customer_tenure_months: Relationship duration
avg_monthly_transactions: Usage frequency
customer_lifetime_value: Total customer value

Credit & Lending

interest_rate: Loan interest rate
loan_amount / loan_tenure / emi_amount: Loan details
default_flag: Loan default indicator
credit_score: Creditworthiness

Risk & Fraud Analytics

fraud_flag: Fraud indicator
fraud_type: Type of fraud
risk_score: Risk rating
fraud_detection_time_sec: Detection time
chargeback_flag: Chargeback indicator
dispute_status: Dispute resolution status

Funding & Strategy

funding_round: Investment stage
funding_amount: Capital raised
investor_type: Investor category
valuation: Company valuation
growth_strategy: Strategic approach
expansion_region: Target region

Product & Engagement

app_sessions: Number of sessions
session_duration_sec: Time spent
bounce_rate: Drop-off rate
feature_used: Product feature
churn_flag: Customer churn indicator
retention_rate: Customer retention

Operational Metrics

app_latency_ms: System latency
api_response_time_ms: API performance
success_rate / failure_rate: Transaction performance
downtime_minutes: System downtime
server_region: Server location

```
